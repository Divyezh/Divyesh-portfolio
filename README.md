## 🌌 About The Project

This is a state-of-the-art personal portfolio website designed for full-stack developers. It distances itself from traditional, static portfolios by implementing a live, interactive 3D particle background, deep purple ambient lighting themes, and complex staggered entrance animations.

### 🌟 Key Features
- **Immersive 3D Space Environment:** Live rendering of a dynamic starfield using Three.js and React Three Fiber.
- **Advanced Staggered Animations:** A fully responsive "honeycomb" V-shaped skill grid that cascades in using Framer Motion.
- **Deep Purple Aesthetic:** Glowing components, blackhole hero videos, and soft ambient lighting generated via modern CSS radial gradients.
- **Performance First:** Built on Next.js 14 App Router, heavily utilizing server components where applicable to ensure rapid load times despite heavy visual assets.
- **Fully Responsive:** Perfectly adapts from ultrawide desktop monitors down to mobile screens.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have Node.js installed (v18.x or higher recommended).
```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Divyezh/Divyesh-portfolio.git
   ```
2. Install NPM packages
   ```sh
   cd Divyesh-portfolio
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🎨 Project Architecture

```text
├── app/                  # Next.js 14 App Router pages and layouts
├── components/           
│   ├── main/             # Top-level sections (Hero, Skills, Projects, Navbar, etc.)
│   └── sub/              # Reusable sub-components (Project Cards, Skill Text, etc.)
├── constants/            # Centralized data store (Skill lists, Project info, Socials)
├── public/               # Static assets (SVGs, Videos, Images)
└── tailwind.config.ts    # Custom deep-purple theme configurations
```
